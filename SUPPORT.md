# Support

ComposeSharp is an open-source project maintained through GitHub.

## Questions and usage help

Use GitHub Discussions for usage questions, design discussion, and roadmap topics:

https://github.com/GaTTGeng/ComposeSharp/discussions

## Bugs and compatibility gaps

Open an issue with a minimal Compose file, Docker Engine version, ComposeSharp version, expected behavior, actual behavior, and relevant logs:

https://github.com/GaTTGeng/ComposeSharp/issues

## Security issues

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).
