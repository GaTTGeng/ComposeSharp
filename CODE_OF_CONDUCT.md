# Contributor Covenant Code of Conduct

ComposeSharp contributors are expected to provide a welcoming, harassment-free experience for everyone. Be respectful, assume good intent, and focus discussion on the work.

Unacceptable behavior includes harassment, discrimination, personal attacks, or publishing private information without permission. Report concerns privately through the repository's security advisory flow or maintainer contact channel. Maintainers may remove, edit, or reject contributions that do not follow this code of conduct.
